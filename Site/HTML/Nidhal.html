Le fichier programme TetrisJeu commence par une fonction qui va permettre de générer aléatoirement un des Tetriminos possible grâce à la commande random(). Vous trouverez d'avantage d'explication à propos du module CLASSES dans l'onglet Zacharie.


L'une des variables les plus essentielles dans la réalisation de ce menu est sans aucun doute le temps.

time1 = pygame.time.get_ticks() (l.172)

La variable time1 donne le temps instantanné depuis le lancement du programme.


time = pygame.time.get_ticks() (l.37)

La variable time quant à elle va nous servir de repère afin de faire clignoter un bouton.



La fonction timer():

def timer(time, time1):

if time1<time+500:

z=1

else:

z=2

return(z)


Cette fonction permet de faire clignoter un bouton lorsqu'il est séléctionner par l'utilisateur.




Le clignotement est en réalité une succession de deux images placées dans une liste, affichées successivement à intervalles de temps de une demi-seconde entre deux images.


if nbr_touch == 1:

screen.blit(quit_list[timer(time, time1)], bt2)

if time+1500<time1:

time=time1

Comme on peut le voir dans l'exemple, la variable time va permettre d'indiquer au bout de combien de temps l'image est affichée. Si cette durée dépasse de plus de une seconde et demi, la variable time prend la valeur de time.



Chaque image représentant un bouton possède 3 modèle:

    • x1>celui lorsque le bouton n'est pas séléctionné ( standart )

    • x2>lorsque le bouton est séléctionné ( gras, souligné et transparent ). Ces images sont placées dans une liste


La séléction d'un bouton s'effectue avec la variable nbrtouch. Chaque bouton est associé à un numéro:

      • Play: nbrtouch = 1

      • Option: nbrtouch = 2

      • Quitter: nbrtouch = 3



Dans le menu option, le principe est le même avec:

      • Musique: nbrtouch = 1

      • Fond d'écran: nbrtouch = 2


Deux autres variables permettent de choisir horisontallement:

      • Musique: choix1

      • Fond d'écran: choix2


Les variables de type 'choix' sont utilisées afin de déterminer les musiques et les fonds d'écran utilisés pendant le jeu grâce à des listes.
